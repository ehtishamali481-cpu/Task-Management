# 🚀 Next.js Task Management System

A full-stack **Task Management Web Application** built with **Next.js 14, MongoDB, and JWT Authentication**.  
This application allows users to create, manage, and track tasks with **role-based access control (Admin & User)** and a modern dashboard.

---

## 📌 Features

✔ User Authentication (JWT Based Login)  
✔ Role Based Access (Admin / User)  
✔ Create, Update and Delete Tasks  
✔ Task Status Management (Pending / Completed)  
✔ Dashboard with Task Statistics  
✔ Secure API Routes  
✔ MongoDB Database Integration  
✔ Responsive UI Design  

---

## 🛠 Tech Stack

### Frontend
- Next.js 14
- React.js
- CSS / Styled Components

### Backend
- Next.js API Routes
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt.js
